1. Extract the zip footlocker.zip file.
2. After that, install local server (we have used Xampp server).
3. The folder footlocker which we have extracted, place the extracted folder into the server's root directory named "htdocs".
4. Start local server, In our case Xampp.
5. Open any browser.
6. open the link is: http://localhost/phpmyadmin/ a)create new database b)then import database footlocker.sql from the folder you placed to "htdocs" directory   
7. Open the link there: http://localhost/footlockershop/index.php
8. Browse to different pages according to your wish.
